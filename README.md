# 屏蔽网站的Python脚本与[hosts文件](https://github.com/Chenghow/net-forbidden/blob/master/etc/hosts)
netlock.py:锁网

unlock.py:放网

通过修改hosts的方式使国内的主流网站无法访问

**脚本仅支持windows系统，请勿使用代理软件**
# 它有什么用？
> 对于大部分用户而言，它***毫无作用***

对于少数群体：
- **工作、学习电脑管理**
- **特殊时间禁娱**
- **可用于戒网** 
- **屏蔽国内服务，适应海外生活**
- *请谨慎用于恶作剧*

你可以提交你的修改
